Sistema de cadastro e listagem de lives, informando quais já foram realizadas ou que ainda irão acontecer pela data cadastrada no sistema.

Tecnologias utilizadas:
 1. API:
	- MongoDB
	- Spring Boot
2. Front-End:
	- Angular (com Material UI)
